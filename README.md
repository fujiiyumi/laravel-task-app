# Task App

Laravelで作成したタスク管理アプリです。

## 機能一覧

- ユーザー登録 / ログイン
- タスク作成
- タスク編集
- タスク削除
- タスク検索
- ステータス管理（未着手 / 進行中 / 完了）
- ページネーション
- 認可機能（他ユーザーのタスク編集防止）

---

## 使用技術

- PHP
- Laravel
- MySQL
- Blade
- Laravel Breeze

---

## 工夫した点

- ログインユーザーごとにタスクを管理できるようにしました
- 検索条件を保持したままページ移動できるようにしました
- 認可処理を共通化し、コード重複を減らしました
- `when()` を使用して検索処理を整理しました
